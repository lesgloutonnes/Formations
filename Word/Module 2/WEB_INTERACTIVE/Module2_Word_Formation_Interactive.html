Partie 1 — Personnaliser Word
Exercice 1
Adapter Word à sa façon de travailler
📚 Théorie
Police et marges par défaut
  • Accueil → lanceur de la boîte Police → Définir par défaut.
  • Disposition → Marges → Marges personnalisées → Définir par défaut.
  • Les réglages s'enregistrent dans le modèle Normal.dotm.
📚 Théorie
Barre d'accès rapide, ruban, raccourcis
  • Barre d'accès rapide : clic droit sur une commande → Ajouter.
  • Personnaliser le ruban : créer un onglet et un groupe sur mesure.
  • Raccourcis clavier : Fichier → Options → Personnaliser le ruban → Raccourcis.
📚 Théorie
Correction automatique & QuickPart
  • Correction automatique : chc → CHC - Service Formation.
  • Blocs QuickPart : sélectionner un contenu → Insertion → QuickPart → Enregistrer la sélection.
  • Réutilisable à tout moment dans tous vos documents.

Gain de temps

Signature, en-tête, mentions légales : un QuickPart évite de tout retaper.

📚 Théorie
Traduction et export PDF
  • Révision → Traduire : sélection ou document entier.
  • Fichier → Exporter → Créer un document PDF/XPS.
  • Ou Enregistrer sous → type PDF : format figé, idéal pour diffuser.

Partie 2 — Automatiser la présentation
Exercice 2
Styles, jeux de styles et modèles
📚 Théorie
Repérer les mises en forme répétitives
  • Même format appliqué à la main plusieurs fois = candidat à un style.
  • Symptômes : gras + taille manuels, retraits refaits à chaque paragraphe.
  • Le volet Styles révèle la cohérence (ou non) du document.
📚 Théorie
Créer, appliquer, enchaîner les styles
  • Nouveau style : volet Styles → bouton Nouveau style.
  • Style suivant : enchaîner Titre → Normal automatiquement.
  • Modifier un style met à jour tout le document.
  • Styles rapides : accès direct dans la galerie Accueil.
📚 Théorie
Aspect global & modèles
  • Jeu de styles et thème : changer l'aspect global d'un coup.
  • Enregistrer comme modèle : Fichier → Enregistrer sous → type .dotx.
  • Un modèle = styles + mise en page + contenus prêts à réutiliser.

S'organiser

Un modèle d'équipe garantit des documents homogènes sans effort.

Partie 3 — Construire un document structuré
Exercice 3
Listes hiérarchisées, titres numérotés, sommaire
📚 Théorie
Styles de listes
  • Listes à puces, numérotées et hiérarchisées (à plusieurs niveaux).
  • Tab / Maj+Tab pour descendre ou remonter d'un niveau.
  • Créer un style de liste pour réutiliser la même présentation.
📚 Théorie
Titres numérotés automatiquement
  • Styles hiérarchiques Titre 1 / Titre 2 / Titre 3 définissent le plan.
  • Liste à plusieurs niveaux liée aux styles de titres → numérotation 1, 1.1, 1.2.
  • Sommaire automatique : Références → Table des matières.
📚 Théorie
En-têtes et pieds de page
  • Double-clic en haut/bas de page pour éditer.
  • Première page différente et pages paires/impaires.
  • Champs : numéro de page, titre du document, date.
Piège : numéroter à la main au lieu d'insérer un champ « Numéro de page ».

Partie 4 — Intégrer des illustrations
Exercice 4
Images, SmartArt, objets Excel, équations
📚 Théorie
Image, WordArt, SmartArt
  • Insérer et modifier une image (outils Image : rogner, styles).
  • WordArt pour un titre décoratif.
  • SmartArt pour illustrer un processus ou une organisation.
📚 Théorie
Tableau et graphique Excel
  • Copier dans Excel → Collage spécial dans Word.
  • Coller avec liaison : le document se met à jour si Excel change.
  • Coller comme image si les données ne doivent pas évoluer.

Lien Word ↔ Excel

La liaison évite de recopier des chiffres et garde tout cohérent.

📚 Théorie
Symboles, équations, habillage
  • Insertion → Symbole et Équation.
  • Habillage : aligné, carré, rapproché, derrière/devant le texte.
  • Maîtriser le positionnement : ancre, alignement, position fixe sur la page.

Partie 5 — Maîtriser les tableaux
Exercice 5
Dessiner, calculer, convertir, colonnes
📚 Théorie
Dessiner, fusionner, fractionner
  • Dessiner un tableau : outils stylo et gomme.
  • Fusionner des cellules (Disposition → Fusionner).
  • Fractionner une cellule ou un tableau.
📚 Théorie
Formules de calcul
  • Disposition → Formule : =SUM(ABOVE), =AVERAGE(LEFT).
  • Mettre à jour le résultat avec F9.
  • Pour des calculs complexes, préférer un objet Excel.
📚 Théorie
Convertir & colonnes journal
  • Convertir texte en tableau (séparateur) et l'inverse.
  • Utiliser un tableau (sans bordures) pour faciliter la mise en page.
  • Disposition → Colonnes : présentation façon journal.
