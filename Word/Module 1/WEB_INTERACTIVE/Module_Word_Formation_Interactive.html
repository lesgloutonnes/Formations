Organisation de la session

Pour 6 à 8 participants

  • Flèches ← → pour naviguer dans le cours.

📚 Théorie
Cas CHC : procédure interne

Quand l'utiliser ?

  • Décrire une procédure, un processus ou des étapes à suivre.
  • Structure : titres, sous-titres, tableau de suivi.
  • Exemple : Procédure interne - Accueil d'un nouveau collaborateur
📚 Théorie
Cas CHC : compte rendu

Quand l'utiliser ?

  • Retranscrire une réunion ou un point d'équipe.
  • Structure : objectif, décisions, actions à suivre.
  • Tableau utile : Décision / Responsable / Échéance.
📚 Théorie
Cas CHC : note interne et document à relire

Note interne

  • Document court, 1 à 2 pages.
  • Pas de table des matières nécessaire.
  • Objectif : clair, propre, prêt à envoyer.

Document à relire

  • Texte existant à corriger ou valider.
  • Commentaires + suivi des modifications.
  • Version finale sans traces de révision.
📚 Théorie
Partie 1 — Structurer avec les styles
⏱️ 45 min · Théorie + Exercice 1
Le réflexe le plus important de la formation
📚 Théorie
Word n'est pas une machine à écrire
  • Un document = des paragraphes, pas des lignes isolées.
  • Entrée crée un nouveau paragraphe.
  • Les styles changent tout le document en une fois.
Piège : titres en gras + taille manuelle au lieu de Titre 1 / Titre 2.
📚 Théorie
Les styles essentiels
  • Normal — texte courant.
  • Titre 1 — grandes parties.
  • Titre 2 — sous-parties.

À retenir

On applique un rôle au texte. Word assure la cohérence.

📚 Théorie
Volet de navigation

Votre meilleur allié

  • Affichage → cocher Volet de navigation.
  • Affiche la structure réelle du document.
  • Permet de repérer immédiatement un titre mal stylé.
📚 Théorie
Modifier un style proprement
  • Clic droit sur un style → Modifier... = méthode recommandée.
  • Mettre à jour le style = rapide mais moins contrôlé.
  • Supprimer de la galerie = à éviter en formation.
🎯 Démo
Démonstration : modifier Titre 1

2 minutes — montrez, puis ils reproduisent

  1. Appliquer Titre 1 à 2 titres.
  2. Clic droit sur Titre 1 → Modifier....
  3. Changer couleur ou taille → OK.
  4. Montrer le volet de navigation.

Partie 2 — Table des matières
⏱️ 25 min · Exercice 2
Automatique grâce aux styles Titre 1 et Titre 2
📚 Théorie
Table des matières automatique
  • Se base sur les styles Titre 1, Titre 2.
  • Références → Table des matières → modèle automatique.
  • Mise à jour : clic droit → Mettre à jour le champ ou F9.
🎯 Démo
Démonstration : insérer et mettre à jour
  1. Placer le curseur après le titre principal.
  2. Références → Table des matières.
  3. Ajouter une sous-partie → mettre à jour toute la table.

Partie 3 — Mise en page
⏱️ 35 min · Exercice 3
Page de garde, en-têtes, numérotation
📚 Théorie
Sauts de page et sections
  • Insertion → Saut de page : nouvelle page simple.
  • Disposition → Sauts → section : changer la mise en page d'une partie.
  • Document simple → privilégier le saut de page.
📚 Théorie
En-têtes et numérotation
  • Insertion → En-tête / Pied de page / Numéro de page.
  • Première page différente : pas de numéro sur la page de garde.
  • Double-clic en haut ou bas de page pour éditer.
Piège classique : numérotation visible sur la page de garde.
🎯 Démo
Démonstration : en-tête et numérotation
  1. Ajouter une page de garde simple.
  2. Insertion → Saut de page.
  3. En-tête Procédure interne + numéro en bas de page.
  4. Cocher Première page différente.

Enrichir sans casser la mise en page

Partie 5 — Réviser et exporter
⏱️ 20 min · Exercice 5
📚 Théorie
Réviser avant diffusion
  • Révision → Commentaire : poser une question.
  • Suivi des modifications : corriger le fond du texte.
  • Fichier → Enregistrer sous → PDF.
  • Désactiver le suivi avant la version finale.
🎯 Démo
Démonstration : commentaire et PDF
  1. Sélectionner un passage → Nouveau commentaire.
  2. Activer le suivi → modifier une phrase → accepter/refuser.
  3. Fichier → Enregistrer sous → PDF.

  • PDF non relu — mise en page cassée à l'envoi.
