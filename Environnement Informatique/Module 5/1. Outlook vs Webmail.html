1/6

Outlook vs Webmail

Comprendre les différences

📧 Outlook (Application)

  • ✓ Application installée sur votre PC
  • ✓ Fonctionnalités avancées
  • ✓ Gestion du calendrier intégrée
  • ✓ Travail hors ligne possible
  • ✓ Règles et filtres avancés
  • ✓ Synchronisation avec d'autres outils Microsoft

🌐 Webmail (Navigateur)

  • ✓ Accessible depuis n'importe quel poste
  • ✓ Pas d'installation nécessaire
  • ✓ Interface simplifiée
  • ✓ Accès rapide et pratique
  • ✓ Idéal pour consultation rapide
  • ✓ Fonctionne sur tous les systèmes

💡 Conseil : Utilisez Outlook pour un travail quotidien intensif et Webmail pour une consultation rapide ou depuis un autre poste.

2/6

Gérer les rendez-vous

Calendrier et réunions

  • Créer un rendez-vous dans Outlook Cliquez sur "Nouveau rendez-vous" dans le ruban Calendrier. Remplissez les informations : titre, date, heure, participants, lieu. Cliquez sur "Envoyer" pour inviter les participants.
  • Voir votre calendrier Dans Outlook, cliquez sur l'icône Calendrier en bas à gauche. Vous verrez votre calendrier avec vos rendez-vous, réunions et disponibilités.
  • Accepter/Refuser une invitation Lorsque vous recevez une invitation par email, vous avez le choix entre : ✓ Accepter, ? Provisoire, ✗ Refuser ou Proposition d'un nouvel horaire. Votre réponse sera envoyée à l'organisateur.
  • Modifier un rendez-vous Double-cliquez sur le rendez-vous dans votre calendrier pour le modifier. Si vous êtes l'organisateur, les modifications seront envoyées aux participants.
3/6

Réunions Webex

Organiser des réunions Webex

  • Choix du type de réunion : Webex ou Outlook Avant de planifier une réunion, vous devez choisir quel planificateur utiliser. Par défaut, Webex est sélectionné, mais vous pouvez opter pour Microsoft Outlook.
    Comment configurer :
    1. Ouvrez l'application Webex
    2. Cliquez sur l'icône Paramètres (roue dentée) en bas à gauche
    3. Accédez à la section Réunions > Programmer les réunions
    4. Choisissez votre planificateur préféré : Webex ou Microsoft Outlook
  • Créer ou rejoindre une réunion Dans Webex, cliquez sur l'onglet Réunions (icône calendrier) dans le menu latéral. Vous accédez à la page des réunions avec trois options :
    1. Démarrer une réunion Webex — Lance immédiatement une réunion instantanée
    2. Se joindre à une réunion — Permet de rejoindre une réunion existante en entrant le numéro ou le lien
    3. Programmer une réunion — Ouvre l'assistant de création de réunion. Selon le choix effectué dans les Paramètres (section précédente), cela ouvrira l'assistant de planification de Webex ou de Microsoft Outlook

Règles et dossiers

Organiser votre messagerie

  • Créer des dossiers Clic droit sur votre boîte de réception > Nouveau dossier. Donnez un nom clair (ex: "Projets", "Urgent", "Archives"). Organisez vos emails en les glissant-déposant dans les dossiers.
  • Créer des règles automatiques Fichier > Règles et alertes > Nouvelle règle. Par exemple : "Déplacer les emails de [expéditeur] vers le dossier [nom]". Les règles s'appliquent automatiquement aux nouveaux emails.
  • Règles utiles Déplacer les newsletters dans un dossier dédié, marquer comme lu les emails de certains expéditeurs, transférer automatiquement certains emails à un collègue.
  • Gérer les règles existantes Dans "Règles et alertes", vous pouvez modifier, désactiver ou supprimer vos règles. Testez-les avant de les activer définitivement.

Tâches, suivis et indicateurs

Ne rien oublier grâce à Outlook

  • Marquer un email avec un indicateur (flag) Cliquez sur le petit drapeau à droite d'un email pour le marquer comme "À suivre". Clic droit sur le drapeau > "Assurer un suivi" pour choisir une échéance : Aujourd'hui, Demain, Cette semaine, Semaine prochaine ou une date personnalisée. Outlook vous enverra une notification à l'échéance.
  • Classer un email Clic droit sur un email > Catégoriser pour lui attribuer une catégorie de couleur (ex: rouge pour urgent, bleu pour projets). Vous pouvez aussi glisser-déposer un email dans un dossier pour le ranger.
  • Créer une tâche Double cliquez sur l'icône "Tâches" (coche) en bas à gauche d'Outlook. Cliquez sur "Nouvelle tâche" pour créer un rappel avec un titre, une date d'échéance et une priorité (haute, normale, basse). Un survol de cette icône vous donne la liste des tâches et des mails avec "Flag"
  • Transformer un email en tâche Glissez-déposez un email sur l'icône "Tâches" en bas à gauche. Outlook crée automatiquement une tâche avec le contenu de l'email. Vous pouvez ensuite modifier la date d'échéance et la priorité.

💡 Astuce : Utilisez les flags pour un suivi rapide et les tâches pour des actions plus complexes nécessitant une planification.

6/6

Signatures et réponses automatiques

Personnaliser vos emails

  • Créer une signature Fichier > Options > Courrier > Signatures. Cliquez sur "Nouveau" et créez votre signature avec votre nom, fonction, coordonnées. Vous pouvez ajouter une image (logo CHC).
  • Signature par défaut Définissez une signature par défaut pour les nouveaux messages et une autre pour les réponses. Vous pouvez avoir plusieurs signatures et choisir celle à utiliser.
  • Réponses automatiques (absence dubureau) Fichier > Réponses automatiques. Activez les réponses automatiques quand vous êtes absent. Définissez les dates et le message à envoyer.
  • Règles de réponse Créez des règles pour répondre automatiquement à certains types d'emails (ex: accusé de réception pour les demandes importantes). Utilisez cette fonction avec précaution.

💡 Astuce : Vérifiez régulièrement vos règles et signatures pour vous assurer qu'elles fonctionnent correctement et sont à jour.
