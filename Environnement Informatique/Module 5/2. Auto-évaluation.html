Outlook vs Webmail

Question 1/6
Où créer une signature dans Outlook ?
Réponse correcte : Fichier > Options > Courrier > Signatures. C'est le chemin pour accéder aux paramètres de signature dans Outlook.
Question 2/6
Comment créer un rendez-vous dans Outlook ?
Réponse correcte : Cliquer sur "Nouveau rendez-vous" dans le ruban Calendrier. C'est la méthode standard pour créer un rendez-vous dans Outlook.
Question 3/6
Comment créer une règle automatique dans Outlook ?
Réponse correcte : Fichier > Règles et alertes > Nouvelle règle. Les règles permettent d'automatiser la gestion de vos emails.
Question 4/6
Comment choisir le planificateur de réunions (Webex ou Outlook) dans l'application Webex ?
Réponse correcte : Paramètres (roue dentée) > Réunions > Programmer les réunions. Vous pouvez y choisir entre Webex ou Microsoft Outlook comme planificateur de réunions.
Question 5/6
Comment marquer un email pour le suivre dans Outlook ?
Réponse correcte : Cliquer sur le drapeau (flag) à droite de l'email pour le marquer comme "À suivre". Vous pouvez ensuite clic droit sur le drapeau > "Assurer un suivi" pour choisir une échéance.
Question 6/6
Quel est l'avantage principal d'Outlook par rapport à Webmail ?
Réponse correcte : Fonctionnalités avancées, gestion du calendrier intégrée, travail hors ligne. Outlook offre plus de fonctionnalités pour un usage professionnel intensif.
