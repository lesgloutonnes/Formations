📝 Auto-évaluation - Module 3

Imprimantes et périphériques

Question 1/5
Quel logiciel faut-il utiliser pour ajouter une imprimante sur votre poste ?
Réponse correcte : Install My Printer, via le raccourci sur le bureau. C'est le seul outil à utiliser pour ajouter ou supprimer une imprimante au CHC.
Question 2/5
Lors de l'ajout d'une imprimante, que faut-il penser à faire ?
Réponse correcte : Cocher « Définir comme imprimante par défaut ». Cela permet que vos impressions soient automatiquement envoyées vers cette imprimante sans devoir la sélectionner à chaque fois.
Question 3/5
Comment supprimer une imprimante de votre poste ?
Réponse correcte : Via Install My Printer : sélectionner l'imprimante puis cliquer sur Supprimer. L'ajout et la suppression se font exclusivement via ce logiciel.
Question 4/5
L'imprimante souhaitée n'apparaît pas dans la liste d'Install My Printer. Que vérifier en premier ?
Réponse correcte : Vérifier que la version d'Install My Printer est au minimum v3.3. Si la version est correcte et que l'imprimante n'apparaît toujours pas, contactez le Service Desk.
Question 5/5
Votre clavier ou souris USB ne fonctionne plus. Quelles sont les étapes à suivre dans l'ordre ?
Réponse correcte : Vérifier les LEDs → Tester un autre port USB → Redémarrer le PC → Service Desk. Il faut suivre ces étapes dans l'ordre avant de contacter le support.
