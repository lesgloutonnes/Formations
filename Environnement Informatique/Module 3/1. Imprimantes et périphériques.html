Ajouter une imprimante

Utiliser Install My Printer

  • 1 Ouvrir Install My Printer Double-cliquez sur le raccourci Install My Printer présent sur votre bureau Windows.
  • 2 Sélectionner l'imprimante Dans le menu déroulant Imprimante, sélectionnez ou tapez le nom de l'imprimante souhaitée. L'emplacement et le pilote s'afficheront automatiquement.
  • 3 Définir comme imprimante par défaut Cochez la case « Définir comme imprimante par défaut ». Cette option est recommandée pour votre imprimante principale afin qu'elle soit automatiquement sélectionnée lors de chaque impression.
  • 4 Ajouter Cliquez sur le bouton Ajouter pour lancer l'installation. L'imprimante sera alors disponible sur votre poste.
⚠️ Bonne pratique : Pensez toujours à cocher « Définir comme imprimante par défaut » lors de l'installation de votre imprimante principale. Cela vous évitera de devoir la sélectionner manuellement à chaque impression.
3/5

Supprimer une imprimante

Retirer une imprimante du poste

  • 1 Ouvrir Install My Printer Comme pour l'ajout, ouvrez le logiciel Install My Printer via le raccourci sur le bureau.
  • 2 Sélectionner l'imprimante à supprimer Dans le menu déroulant Imprimante, sélectionnez le nom de l'imprimante que vous souhaitez retirer.
  • 3 Cliquer sur Supprimer Cliquez sur le bouton Supprimer pour retirer l'imprimante de votre poste.
⚠️ Attention : Si vous supprimez votre imprimante par défaut, pensez à en redéfinir une autre comme imprimante par défaut pour éviter les erreurs d'impression.

💡 Rappel : L'ajout et la suppression d'imprimantes se font exclusivement via Install My Printer. N'utilisez pas les paramètres Windows pour gérer vos imprimantes.

Dépannage impression

Résoudre les problèmes courants

  • L'imprimante n'apparaît pas dans la liste
    1. Demandez d'abord à installer l'imprimante via Install My Printer
    2. Si le nom de l'imprimante n'apparaît pas dans la liste déroulante, vérifiez la version du logiciel : elle doit être au minimum v3.3
    3. Si l'imprimante n'est toujours pas dans la liste des imprimantes installables, contactez le Service Desk
  • L'impression ne sort pas Vérifiez que la bonne imprimante est sélectionnée (celle par défaut ou celle choisie manuellement). Vérifiez également que l'imprimante est allumée et qu'il n'y a pas de bourrage papier ou de manque de papier/toner.
  • La file d'impression est bloquée Si vos documents restent en attente, essayez de supprimer l'imprimante via Install My Printer puis de la réajouter. Si le problème persiste, contactez le Service Desk.
  • Commande de cartouche d'encre ? Notre prestataire MigDP s'occupe de fournir automatiquement et à l'avance les consommables.

Périphériques USB

Micro, clavier, souris : les règles de base

Votre périphérique USB ne fonctionne pas ? Suivez ces étapes dans l'ordre avant de contacter le Service Desk :

  • 1 Vérifier les LEDs du périphérique Si le périphérique possède des voyants lumineux (LEDs), vérifiez qu'ils sont allumés. Un voyant éteint peut indiquer un problème d'alimentation ou de connexion.
  • 2 Tester un autre port USB Débranchez le périphérique et essayez un autre port USB : testez les ports en facade et à l'arrière du PC. Un port défectueux peut être la cause du problème.
  • 3 Redémarrer le PC Si le périphérique n'est toujours pas reconnu, effectuez un redémarrage complet du PC. Cela permet de réinitialiser les pilotes USB.
  • 4 Contacter le Service Desk Si après toutes ces vérifications le périphérique ne fonctionne toujours pas, contactez le Service Desk en précisant le type de périphérique, les étapes déjà tentées et le comportement observé.
