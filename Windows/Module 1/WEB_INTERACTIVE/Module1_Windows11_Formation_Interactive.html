📚 Théorie
Le bureau Windows 11
  • Barre des tâches centrée : boutons regroupés au milieu (Démarrer, Recherche, applications épinglées).
  • Menu Démarrer : grille d'applications épinglées + Recommandations (plus de vignettes/tuiles dynamiques).
  • Widgets (⊞ Win+W) : météo, actualités, agenda à gauche de l'écran.

Changement clé vs Windows 10

Plus de vignettes dynamiques : on épingle des applications et on les regroupe en dossiers.

Partie 1 — Naviguer dans Windows
⏱️ 40 min · Exercice 1
Lancer/fermer un programme, Snap Layouts, raccourcis
📚 Théorie
Exécuter et fermer un programme
  • Ouvrir : Démarrer (centré) → application épinglée ou Toutes les applications.
  • Fermer : croix ✕ en haut à droite, ou Alt+F4.
  • Programme bloqué : Gestionnaire des tâches (Ctrl+Maj+Échap).
📚 Théorie
Manipuler les fenêtres — Snap Layouts
  • Réduire / Agrandir / Restaurer avec les boutons en haut à droite.
  • Snap Layouts : survoler le bouton Agrandir (ou ⊞ Win+Z) → choisir une disposition.
  • Ancrage clavier : ⊞ Win+← / →.
  • Bureaux virtuels : ⊞ Win+Tab (Affichage des tâches).
📚 Théorie
Écran tactile & raccourcis

Écran tactile (sans « mode tablette »)

  • Windows 11 s'adapte automatiquement : icônes plus espacées sans clavier.
  • Gestes : balayer depuis les bords, 3 doigts pour changer de fenêtre.
  • Pas d'interrupteur « Mode tablette » comme dans Windows 10.

Créer un raccourci

  • Clic droit fichier/programme → Afficher plus d'options → Créer un raccourci.
  • Glisser une icône sur le Bureau.
  • ⊞ Win+E (Explorateur).
🎯 Démo
Démonstration : Snap Layouts
  1. Ouvrir l'Explorateur et le navigateur.
  2. Survoler le bouton Agrandir d'une fenêtre → choisir une disposition à 2.
  3. Cliquer la seconde zone pour y placer l'autre fenêtre.
  4. Montrer ⊞ Win+Z et un second bureau.

Naviguer et gérer les fenêtres

Objectif : ouvrir, organiser et fermer des programmes avec aisance.

⏱️ 20 minutes

✏️ Exercice 1
Naviguer — étapes 1 à 4
  1. Ouvrez l'Explorateur, le navigateur et la Calculatrice depuis Démarrer.
  2. Utilisez Snap Layouts (⊞ Win+Z) pour placer 2 fenêtres.
  3. Réduisez tout (⊞ Win+D) puis restaurez.
  4. Basculez entre les fenêtres avec Alt+Tab.
✏️ Exercice 1
Naviguer — étapes 5 à 7
  1. Créez un second bureau (⊞ Win+Ctrl+D).
  2. Créez un raccourci de la Calculatrice sur le Bureau.
  3. Fermez la Calculatrice avec Alt+F4.
✅ Solution 1
Correction : navigation
  1. Deux fenêtres placées via une disposition Snap Layouts.
  2. Un second bureau est visible dans ⊞ Win+Tab.
  3. Le raccourci Calculatrice apparaît sur le Bureau (flèche sur l'icône).
  4. Réflexe : ⊞ Win ouvre Démarrer et lance une recherche immédiate.
⚠️ Pièges
Erreurs fréquentes — navigation
  • Chercher le « Mode tablette » : il n'existe plus dans Windows 11.
  • Ignorer Snap Layouts et redimensionner les fenêtres à la main.
  • Forcer l'arrêt du PC au lieu de fermer le programme bloqué.
  • Confondre raccourci (flèche) et fichier d'origine.

Partie 2 — Personnaliser son espace
⏱️ 40 min · Exercice 2
Menu Démarrer, applications épinglées, barre des tâches, Widgets
📚 Théorie
Le menu Démarrer (Windows 11)
  • Zone Épinglées en haut, Recommandations en bas (fichiers/apps récents).
  • Épingler : clic droit sur une app → Épingler au menu Démarrer.
  • Détacher : clic droit sur l'icône épinglée → Détacher de Démarrer.
  • Grouper : faire glisser une app sur une autre → crée un dossier que l'on nomme.
  • Réorganiser par glisser-déposer ; bouton Toutes les applications pour la liste complète.
📚 Théorie
Barre des tâches & Widgets
  • Épingler un programme : clic droit sur l'icône → Épingler à la barre des tâches.
  • Clic droit sur la barre → Paramètres de la barre des tâches : afficher/masquer Recherche, Widgets, Affichage des tâches.
  • Alignement des icônes : centré (par défaut) ou à gauche.
  • Widgets (⊞ Win+W) : ajouter/retirer des cartes (météo, agenda…).
📚 Théorie
Bureau, raccourcis et arrière-plan
  • Organiser le Bureau : clic droit → Trier par, regrouper les raccourcis.
  • Repérer les raccourcis vers les principaux programmes (icône avec flèche).
  • Arrière-plan : clic droit Bureau → Personnaliser → Arrière-plan ; thèmes clair/sombre.

À retenir

Épingler ≠ installer. On crée un accès rapide vers un programme déjà installé.

Partie 3 — Dossiers et fichiers
⏱️ 45 min · Exercice 3
L'explorateur : copier, déplacer, renommer, supprimer, récupérer
📚 Théorie
L'explorateur de Windows 11
  • Ouvrir : ⊞ Win+E ou icône dossier de la barre des tâches.
  • Barre d'outils simplifiée : Couper, Copier, Coller, Renommer, Supprimer en icônes.
  • Onglets (comme un navigateur) pour ouvrir plusieurs dossiers dans une fenêtre.
  • Volet gauche : Accueil, Galerie, Ce PC, lecteurs (C:), périphériques, réseau.
📚 Théorie
Copier, déplacer, renommer, supprimer
  • Copier Ctrl+C · Couper Ctrl+X · Coller Ctrl+V.
  • Renommer : F2 ou icône Renommer de la barre d'outils.
  • Supprimer : Suppr (va dans la Corbeille).
  • Récupérer : ouvrir la Corbeille → clic droit → Restaurer.
  • Sélection multiple : Ctrl+clic ou Maj+clic.
🎯 Démo
Démonstration : ranger avec les onglets
  1. Créer un dossier (Ctrl+Maj+N).
  2. Ouvrir le dossier cible dans un second onglet.
  3. Glisser des fichiers d'un onglet à l'autre.
  4. Supprimer puis restaurer depuis la Corbeille.

  1. Supprimez le doublon a_trier - copie (1).txt.
  2. Restaurez-le depuis la Corbeille, puis supprimez-le définitivement.

✅ Solution 3

  1. Chaque fichier est rangé dans le bon sous-dossier (via les onglets).
  2. Fichiers renommés clairement (ex. Photo_equipe_CHC.txt).
  3. Doublon supprimé, restauration testée puis suppression définitive.

Erreurs fréquentes — fichiers
  • Confondre Couper (déplace) et Copier (duplique).
  • Maj+Suppr supprime définitivement (sans Corbeille).
  • Renommer un fichier et effacer son extension (.txt, .docx).
  • Retirer une clé USB sans « Éjecter » pendant une copie.

Partie 4 — L'outil de recherche
⏱️ 25 min · Exercice 4
Tout retrouver : documents, logiciels, e-mails, sites web
📚 Théorie
Rechercher depuis Démarrer
  • Cliquer la Recherche (loupe centrée) ou appuyer sur ⊞ Win et taper.
  • Onglets de résultats : Tout, Applications, Documents, Web, Paramètres.
  • Lancer un programme en tapant son nom (ex. « Word ») puis Entrée.
📚 Théorie
Rechercher dans l'explorateur
  • Champ Rechercher en haut à droite de l'explorateur.
  • La recherche porte sur le dossier (et ses sous-dossiers).
  • Filtres : type, date de modification, taille.
  • Astuce : taper *.docx pour tous les documents Word.

Le bon réflexe

Mieux vaut chercher que parcourir des dizaines de dossiers.

Partie 5 — Utilitaires & PowerToys
⏱️ 25 min · Exercice 5
Programmes préinstallés et outils qui font gagner du temps
📚 Théorie
Les programmes préinstallés
  • Outil Capture d'écran : ⊞ Win+Maj+S (capture + enregistrement vidéo).
  • Paint, Bloc-notes (avec onglets), Calculatrice.
  • Paramètres (⊞ Win+I), interface réorganisée par catégories.
  • Sécurité Windows (antivirus Defender intégré).
📚 Théorie
Les PowerToys

Outils gratuits de Microsoft (à installer)

  • FancyZones : dispositions de fenêtres sur mesure (au-delà des Snap Layouts).
  • PowerRename : renommer des fichiers en masse.
  • PowerToys Run (Alt+Espace) : lanceur rapide.
  • Color Picker, Image Resizer, Text Extractor.
À noter : les PowerToys s'installent (Microsoft Store / GitHub / winget) ; demandez l'accord du service informatique.
